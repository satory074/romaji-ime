# RomajiIME インストール手順（Mac）

ローマ字で打つと **AI が日本語に変換**してくれる入力メソッドです。
モード切替なし・英語混じりもそのまま打てます。

所要 5 分。**無料の Gemini API キー**を 1 つ用意するだけです。

---

## 1. Gemini の無料キーを取る

1. https://aistudio.google.com/apikey をブラウザで開く（Google ログイン）
2. **「Create API key」** をクリック
3. 出てきた `AIza…` で始まる文字列を**コピー**（あとで貼ります）

> 無料枠で使えます。※無料枠の入力は Google の品質改善に使われ得るので、
> パスワードなど機微な文章の変換は避けてください。

## 2. インストール

1. 受け取った `RomajiIME-macos.zip` を**ダブルクリックで展開**
2. 展開した `RomajiIME-macos` フォルダを開く
3. **「ターミナル」アプリ**を開き、次を打ってから（最後にスペース）、
   フォルダ内の `install.sh` を**ドラッグ＆ドロップ**して Enter:

   ```
   bash 
   ```

   （`bash /…/RomajiIME-macos/install.sh` のようになって実行されます）
4. 画面の指示に従い、**手順1でコピーしたキーを貼り付け**て Enter
   （貼り付けは ⌘V。キーは画面に表示されません＝正常です）

スクリプトがやること: 検疫属性の解除 → アプリを `~/Library/Input Methods/`
へ配置 → キーを `config.json` に保存（自分だけ読める権限）。

## 3. 有効化（初回のみ）

1. **システム設定 ▸ キーボード ▸ 入力ソース ▸ 「編集…」**
2. **「＋」▸ 日本語 ▸ RomajiIME ▸ 追加**
3. **Ctrl+Space**（または画面右上の入力メニュー）で **RomajiIME** に切り替え

> リストに「RomajiIME」が出てこないときは、一度**ログアウト→ログイン**してください。

## 4. 使ってみる

テキスト編集や任意のアプリで `nihongo` と打って少し待つと、下に候補が出ます。

| 操作 | 結果 |
|---|---|
| そのまま **Enter** | `nihongo`（打った通り。英語や記号はこれ） |
| **Space** → **Enter** | `日本語`（AI 変換を確定） |
| Space を続けて押す | 候補を巡回 |
| 数字キー | その番号の候補を確定 |
| **Esc** | 候補を消す |

英語混じりもいけます: `nihongo wo github de kanri` → `日本語をGitHubで管理`

---

## こまったとき

- **「開発元を確認できないため開けません」** → `install.sh` を使えば検疫は自動解除
  されます。手動なら: システム設定 ▸ プライバシーとセキュリティ で「このまま開く」。
- **変換されない / "変換中…" のまま** → キーが正しいか確認。`install.sh` を再実行して
  入れ直せます。設定は `~/Library/Application Support/RomajiIME/config.json`。
- **アンインストール** → `~/Library/Input Methods/RomajiIME.app` を削除し、
  入力ソースから外す。設定を消すなら `~/Library/Application Support/RomajiIME/` も削除。
